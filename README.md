# prefix_lisp_parser
Prefix Lisp Parser with GUI Graph Functionality for Special Defined Operations

## Built-With
C++ / QT GUI Framework / CATCH Test Framework
Compiled with CMAKE on Windows

## Valid Operations
Constants: pi
Arithmetic Operations:
> Add: +
> Subtract: -
> Multiply: *
> Division: /

Logical Operations:
> not
> and
> or
> '<'
> '>'
> '<='
> '>='
> '='
> 'if'

Other Functions:
> Log Base Ten: Log10
> pow
> sin
> cos
> arctan

GUI Functions:
> point
> line
> arc
